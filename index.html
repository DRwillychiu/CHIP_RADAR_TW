📊 損益計算方式（重要）

本系統計算三種損益，請依需求解讀：

① 當日沖銷損益（Intraday P&L）

當日沖銷損益 = (賣均 - 買均) × min(買張, 賣張) × 1000 / 10000 萬

意義：同一天內，分點「當沖」部分的實現損益。

⚠️ 不含留倉部位，會低估當日實際獲利（若分點當日大量買進但沒賣）。

② 累積已實現損益（Realized P&L, 自基準日起算）

採 FIFO（先進先出）部位追蹤： 1. 每日爬取的「買進張數」視為建倉 2. 每日爬取的「賣出張數」按 FIFO 從未平倉部位扣除 3. 扣除時的 (賣均 - 對應建倉成本) × 張數 × 1000 / 10000 即當日實現 4. 累積 = 所有實現損益加總

📅 基準日：— 之前的部位無從追蹤。

📌 期間篩選：日/本週（週一起）/本月（1 號起）/全部累積

③ 未平倉部位（Open Positions）

該分點還沒賣掉的股數 + 加權平均建倉成本，代表分點仍持有的部位。

⚠️ 重要限制：
• 若基準日前已有留倉未追蹤，計算會偏差
• 富邦僅顯示當日前 30 名買賣超，超出排行的交易追蹤不到
• 張數與金額資料源自富邦頁面，均價 = 金額/張數（與真實成交均價差距極小）

📖 八大分頁功能

  1. 🌅 今日三視角（主頁）：盤後最重要的觀察工具，含三個子視角：
    • 🔥 當沖觀察：今日買賣接近的股票，看誰手感好
    • 🌙 隔日沖觀察：今日建倉但未賣，明日要盯他們怎麼出
    • 🐢 波段觀察：連續多日建倉的分點×個股組合（長期佈局訊號）
  2. 🔥 共買榜：當日被多分點同時買超的個股
    • 💰 金額榜 vs 📦 張數榜 切換（不同排行看到不同個股！）
    • 新增「強度」排序（分點數 × 金額）最能反映共識強度
  3. 📊 分點動態：各分點當日買超（含張數/均價/當沖損益/風格徽章）
  4. 🏆 累積排行：分點/高手/個股 的累積獲利排行
    • 期間：當日 / 近5日 / 本週 / 本月 / 近60日 / 累積全部
  5. 👤 高手合併視圖（新功能，重要）：一位高手有多個分點時，合併看全貌
    • 民哥 2 分點、航海王 6 分點、陳族元 4 分點... 分開看不出全貌
    • 含：總損益 / 今日淨買 / 未平倉 / 風格判定 / 旗下共識個股（多分點共買）
  6. 📈 個股追蹤：跨日歷史進出查詢
  7. 👥 單分點偏好：單一分點近期買進偏好與績效
  8. ⚙️ 關注設定：自訂重點分點清單

🎭 交易風格判定邏輯

系統自動為每檔股票標記風格：

當沖比率 = min(買張, 賣張) / max(買張, 賣張) ≥ 0.7 → 🔥 daytrade (當沖) 0.3-0.7 → ⚖️ partial (部分當沖+留倉) < 0.3 → 🌙 overnight (主要建倉/留倉)

分點級風格 = 當日所有股票的平均當沖比率：

分點平均當沖比 ≥ 0.55 → 🔥 當沖派 (daytrader) 0.30-0.55 → ⚖️ 混合型 (mixed) < 0.30 → 🐢 波段派 (swing)

💡 波段派分點 = 大多建倉留倉 → 明日開盤要看他們怎麼處理手上部位。
