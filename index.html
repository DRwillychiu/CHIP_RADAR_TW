🔒 安全機制說明

本網站使用 AES-256-GCM 加密保護所有分點資料：

  • 資料來自 GitHub Pages 公開倉庫，但 JSON 內容完全加密
  • 密碼不存於程式碼，僅存於 GitHub Secrets（您可掌控）
  • 網站絕不記錄密碼，每次關閉瀏覽器都需重新輸入
  • 解密在您本機瀏覽器中進行，密碼不傳輸到任何伺服器

🤖 自動化機制

每個交易日台灣時間 18:30，GitHub Actions 自動爬取資料並用您的密碼加密後儲存。

📖 功能說明

  1. 共買榜：當日被多分點同時買超的個股
  2. 分點動態：各分點當日買超前 30 名
  3. 個股追蹤：跨日歷史進出查詢
  4. 高手偏好：分點近期持續買進偏好
  5. 關注設定：自訂重點分點清單
